# practica-final-aws-s3

## Descripción
Despliegue de un sitio web estático utilizando AWS S3.

## Servicios utilizados
- Amazon S3
- Static Website Hosting

## Pasos realizados
1. Creación del bucket en Amazon S3.
2. Configuración para permitir acceso público.
3. Subida del archivo index.html e imagen.jpg.
4. Activación de Static Website Hosting.
5. Configuración de Bucket Policy.
6. Verificación del endpoint web.
